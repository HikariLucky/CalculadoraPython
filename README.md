# 🧮 CalculadoraPython

Uma calculadora simples feita em Python que realiza operações matemáticas diretamente no terminal.  
Perfeita pra quem quer praticar lógica de programação e manipulação de dados de forma prática e objetiva.

---

## 🚀 Funcionalidades

- ➕ Adicionar múltiplos valores
- ➕ Somar todos os valores inseridos
- ➖ Subtrair todos os valores inseridos
- ✖️ Multiplicar todos os valores inseridos
- ➗ Dividir todos os valores inseridos  
  *(com tratamento de divisão por zero)*
- 📋 Visualizar os valores digitados
- 💻 Interface simples via terminal

---

## 🛠️ Tecnologias utilizadas

- Python 3.x  
- Biblioteca padrão do Python (`math`)

---

## 📦 Como usar

Clone o repositório:

```bash
git clone https://github.com/HikariLucky/CalculadoraPython.git

Acesse a pasta do projeto:

cd CalculadoraPython

Execute o programa:

python calculadora.py

> 💡 Em alguns sistemas, pode ser necessário usar:



python3 calculadora.py
